# ImmigrationKart - Visa Consultancy Website

A modern, responsive frontend website for ImmigrationKart, a distinguished immigration consultancy firm offering visa and migration services.

## 🌟 Features

- **Responsive Design**: Fully responsive on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Interactive Elements**: 
  - Hamburger navigation menu
  - Smooth scrolling
  - FAQ accordion
  - Contact form with validation
  - Scroll-to-top button
- **Service Information**: Comprehensive details about visa services offered
- **Testimonials**: Client success stories and feedback
- **Contact Integration**: Multiple ways to get in touch
- **Social Media Links**: Connected with all major platforms

## 🔧 Customization

### To change colors:
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #1e3a8a;
    --secondary-color: #06b6d4;
    /* ... other colors ... */
}
```

### To add new services:
Add a new service card in `index.html`:
```html
<div class="service-card">
    <div class="service-icon">
        <i class="fas fa-icon"></i>
    </div>
    <h3>Service Name</h3>
    <p>Service description</p>
</div>
```

### To add new testimonials:
Add a new testimonial card in the testimonials section with the same structure as existing ones.
